Recent and Previous Courses

Computer Science Courses:

CSCI-1101: Introduction to Computing and Problem Solving

    An introduction to computing using various programming languages as examples. This class is required for Computer Science Pathway at Normandale Community College. The class uses Scratch for interactive examples and moves along with C programming language as a support materials.

CSCI-1111: Introduction to Programming in C

    This course focus heavily on C programming language syntax and structure. It's also required for Computer Science Pathway at Normandale Community College.

CSCI-2001: Introduction to Computing and Programming in Python

    This course focuses on Python programming language and common data types. It's required for Computer Science Pathway at Normandale Community College. Over the course, we explore basic data types and express some mathematical problems.

CSCI-2002:Introduction to Algorithms and Data Structures

    This course is required for Computer Science Pathway at Normandale Community College. In this course, we explore basic data structures and algorithms using Java Programming Language. It also introduces some mathematical formulas for analyzing time and space complexities.

CSCI-2021: Machine Architecture and Organization

    This is a required course for Computer Science Pathway at Normandale Community College. This course introduces the architecture and hardware of computer system. We use curcuit board and program in C/C++ for Arduino breadboard. Some works are done in Assembly language.

CSCI-2011: Discrete Structures of Computer Science

    It's a required course for Computer science Pathway at Normandale Community College. In this course, we learn about discrete mathematics use in Computer Science. It introduces basic logics and mathematical proofs to solve problems in Computer Science.

CSCI-4041: Algorithms

    This course is required for Computer Science Program at the University of Minnesota Twin Cities. It focuses deeper on data structures and algorithms in mathematical ways. In contrast to previous data structures and algorithms course, this course uses mathematical proofs instead of coding.

CSCI-2041: Advanced Programming Principles

    This course is required for Computer Science program at the University of Minnesota Twin Cities. It introduces the principles of Functional Programming Paradigm using OCaml to demonstrate.

CSCI-4061: Introduction to Operating Systems

    This is a required course for Computer Science program at the University of Minnesota Twin Cities. Over the course, we learn about the core principles of Operating Systems with both conceptual and practical topics. This course also a project-heavy and labs.

CSCI-3081W: Program Design and Development

    This course is also required for Computer Science program at the University of Minnesota Twin Cities. In this, we have the opportunity to work in a team and explore the design patterns in Software Development. We apply basics Software Engineering Principles and Design to work on Drone Simulation Project (Autonomous Vehicle) using C++ programming language.

CSCI-4511W: Introduction to Artificial Intelligence

    An introduction to AI course that is one the other elective Computer Courses at the University of Minnesota Twin Cities. In this course, we learn a lot about the foundation of Artificial Intelligence and apply knowledge to work on research project. There are many topics covered in this course using mathematical logics and proofs.

CSCI-4011: Formal Languages and Automata Theory

    This course counts for Upper Division Math-Oriented Course (required for BS Computer Science). It introduces the finite state machine, Turing Machine, and many other topics. The topics are very abstract and essential for Theoretical Computer Science background.

CSCI-4921: History of Computing

    This course is also an elective Computer Science course, and it focuses on history of computer and invention of computing machines. It introduces the first human computing projects, computing for war and society, and modern digital world.

CSCI-4131: Internet Programming

    This is an elective Computer Science course at the University of Minnesota Twin Cities. This introduces a wide range of topics in Internet Protocols and Web Programming. Over the course, we begin with creating server in Python, and then move to modern web development using various JavaScript Frameworks.

CSCI-4611: Programming Interactive Computer Graphics and Games

    This course counts as a Computer Science elctive, which introduces the basics mathematics and Graphics Pipeline. Over the course, we learn about algorithms and techniques used in game design and development. We also have the opportunity to work on various projects to implement the features for those techniques.

CSCI-5115: User Interface Design, Implementation, and Evaluation

    This is an elective and also a breadth course Master program in Computer Science at the University of Minnesota Twin Cities. In this course, we have opportunity to work as a team and perform practical work outside of classroom. We interviews audiences and analyze data, and we design features and product to help solve their problems. This course focuses heavily on User Experience, which is very beneficial for researchers and designers.

CSCI-5801: Software Engineering I

    This course is an elective and also a breadth course for Master program in Computer Science at the University of Minnesota Twin Cities. As an advanced introduction to Software Engineering course, we have the opportunity to work in a team using Waterfall and Agile Scrum methodologies. In this course, we work on Voting System that inlcude all stages of Waterfall and move on to Agile Scrum Framework. We write Software Requirements Specfication, Software Design Document, Implementation, Unit Test and System Test.
